# WDD130 - Web Fundamentals

## Course Overview
WDD130 is an introductory web development course covering the fundamentals of web design and development.

## Topics Covered
- HTML basics
- CSS styling and layout
- Responsive web design
- Web accessibility
- Git and version control

This repository contains course materials, assignments, and projects for WDD130.
